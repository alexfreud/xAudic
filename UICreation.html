Creating a new User Interface For xAudic

These instructions show you how to create a new nib based User Interface for xAudic. Although they show the steps to create a UI from scratch, it is much easier to take one of the include UI's and modify it. They can be found in xAudic.app/Contents/Resources/UIPlugins.

  • Download the xAudic interface builder palette from http://www.xaudic.net

    You must copy the palette to /Developer/Palettes.

  • Start InterfaceBuilder and load the palette

    Go to InterfaceBuilder->preferences and add the new palette.

  • Create new empty Cocoa nib
  • Drop drag the "Nib Bases User Interface Controller" (or NibUserInterface) object from the palette into the instances tab.
  • Connect the "window" outlet in your NibUserInterface to the window object
  • Drop a window in
  • Create your gui.

    You can have the following controls in your interface, which should be connected to the given outlet in the NibUserInterface. The controls target action should be set the given action in the NibUserInterface

  • OPTIONAL: Add a VisualizationBox to your UserInterface

    Drag the picture of the Spectrum anylizer from the palette into your window. This a NSBox that will display the active visualiztion. The box can set to any size and it's attributes can be changed freely. Note that although you can make the view any size, some visualization plugins may not be sizable. These visualation will be centered inside the box.

  • OPTIONAL: Add a PlaylistEditor to your UserInterface

    Drag the picture of the PlaylistEditor from the palette into your window.

  • Setup a WindowMovingView if needed.

    If your window is setup to be transparent, you'll need a place for the user to grab on and move it. You can place a WindowMovingView in your nib to accomplish this. Drag the WindowMovingView from the xAudic palette to your window (it's the double arrows)

  • Setup options.

    Click on the NibUserInterface in the instances window and go to Attributes in the Inpector window.

    Current Window Options:

    • No Title Bar - set to selected to have the title bar removed from your window.
    • Click anywhere to move - set to selected to allow the window to be moved by click in anywhere on it
    • Window Background Color - the color to set the window background to. This should be a NSColorWell.
    • Window Opaque - Should the window be opaque?
    • Window Alpha Value - The alpha value for the window.

    Current Transport Button Options:

    • Image Dims When Disabled - Should the button image be dimmed when disabled. (Does not seem to work? AppKit Bug??)
    • Shows Border Only While Mouse Inside - show the border of the button only when the mouse is inside.
    • Highlights By - how the button is highlighed. (see NSButtonCell documentation)
    • Shows State By - how the buttons show there state. (again, see NSButtonCell documentation)
  • Save your nib to one of the following directories.
    • ~/Library/xAudic/Plugins
    • /System/Library/xAudic/Plugins
    • /Network/Library/xAudic/Plugins
    • /Local/Library/xAudic/Plugins
